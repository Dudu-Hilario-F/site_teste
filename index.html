🚀 Laboratório de Estudos Backend & DevOps 🚀

Ambiente de testes para tecnologias de infraestrutura e desenvolvimento backend

🎯 Objetivo do Laboratório:

  • 🐳 Treinar deploy com Docker e containers
  • 🌐 Configurar Nginx como proxy reverso e balanceador
  • ☁️ Implementar soluções em Cloud (AWS, Azure, GCP)
  • ☕ Desenvolver APIs REST com Java Spring Boot
  • 🗄️ Integrar bancos de dados (PostgreSQL, MySQL, MongoDB)
  • ⚖️ Configurar balanceadores de carga

🛠️ Stack Tecnológica:

  • Backend: Java, Spring Boot, Spring Security
  • Banco de Dados: PostgreSQL, MySQL, Redis
  • Infraestrutura: Docker, Kubernetes, Nginx
  • Cloud: AWS EC2, RDS, Load Balancer
  • Monitoramento: Prometheus, Grafana
